JavaScript is one of the most popular programming languages in the world. created in 1995, by Brendan Eich at NetScape.

JavaScript (Js) is now widely used also outside of the browser .The rise of Node js unlocked Backend development. Js is now also used to develop Mobile Apps(React Native) , Embedded applications.

#### Definition

Js is a lightweight , interpreted, Object-oriented language and is best known as the Scripting language for web pages. It is a
Prototype-Based , Multiparadigm Scripting language that is Dynamic and supports 100% object - oriented , imperative and functional programming style.
