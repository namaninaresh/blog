## Binary Number System

The Binary Number System uses only two digits : 0 & 1.The Numbers in this system have a base of 2. Digits 0 and 1 are called bits and 8 bits together make a Byte.

- 0 bit means No or False
- 1 bit means Yes or True.

The Binary Number system uses positional notation. each digit is multiplied by the appropriate power of two based on its position

## Converting a number to Binary

Decimal Number 0-9 are represented in Binary as 0, 1, 10, 11, 100, 101, 110, 111, 1000 and 1001.

#### 1st Process - Remainder and Divisor.

##### Steps :

1. Divide the Number by 2
2. Get the Integer quotient for next iteration.
3. Get the Remainder for the binary digit.
4. Repeat the steps until the quotient is equal to 0.

## Negative Number to Binary

1.Get postive Number Binary of number i.e., (13) = 0000 1101

2.Convert to 1's Complement , exchange 0 to 1 & 1 to 0

0000 1101 ---> 1111 0010

3.Now add 2's Complement , Add 1 to the Binary

```css file=index.css show=false
1  1  1  1  0  0  1  1
```

## Convert Floating Numbers into Binary Numbers

Steps:

1.First find the binary value of Integral part of number i.e., number which left to decimal point .<br></br> 2.Then find the binary value of Fractional part of number i.e., number which right to decimal point.

EX: 4.25
Where 4 is Integral part
and 0.25 is a Fractional part

Integral Part (4)<sub>10</sub> = ( 100)<sub>2</sub>

Fractional Part (0.25)

<strong>Steps :</strong> To Convert the Fractional part to Binary, multiply Fractional
part with 2 and take the one bit which appears before the decimal point.

2.follow the same procedure with after the decimal point (.) part until it becomes 1.0

like

0.25 \* 2 = 0.50 // take 0 & move 0.50 to next step

0.50 \* 2 = 1.00 // take 1 & stop the process.

(0.25)<sub>10</sub> = ( 01 )<sub>2</sub>

Combine both Integral and Fractional

(4.25)<sub>10</sub> = ( 100.01 )<sub>2</sub>
