- Arrays is a collection of similar items stored at contiguous memory locations.
- Elements of the array can be randomly accessed
- The array data structure is not completely dynamic
- Faster in accessed

#### Disadvantages

- Wastage of memory because arrays are fixed in size.
- Time completely increase in insertion & deletion operation
- If there is no enough space in form of contiguous , then In this case you will not able to initialize your array
- size cannot increase once declared
